카카오 알림톡 사용 가이드
알림톡은 카카오톡 채널 개설 → 와이드샷 연동 → 템플릿 등록/승인 절차가 모두 완료되어야 발송 가능합니다. 전체 소요 기간: 약 3~7 영업일

  1. 카카오톡 채널 관리자센터 접속 → 카카오 계정 로그인
  2. 새 채널 만들기 → 채널명, 프로필 이미지, 소개 입력
  3. 채널 개설 완료 후 비즈니스 채널 전환 신청
    • 사업자등록증 사본 제출 필요
    • 심사 기간: 약 1~3 영업일
  4. 비즈니스 채널 승인 완료 확인

  1. 세종네트웍스 담당자에게 알림톡 사용 신청 연락
    • 필요 서류: 사업자등록증, 카카오톡 채널 URL
  2. 세종 측에서 카카오 비즈메시지 허브파트너 계정 연동 처리
  3. 연동 완료 시 senderKey (발신 프로필 키, 40자) 발급
    • 이 키를 서버 환경변수 SEJONG_SENDER_KEY에 설정
    • 또는 관리자에게 전달하여 사용자별 설정
senderKey 예시: abcd1234abcd1234abcd1234abcd1234abcd1234 (40자 영숫자)

  1. 카카오톡 채널 관리자센터 → 비즈메시지 → 알림톡 템플릿 관리
  2. 새 템플릿 등록
    • 템플릿 코드: 자동 발급 (예: SJB_102055)
    • 메시지 유형: 기본형(BA) / 부가정보형(EX) / 채널추가형(AD) / 복합형(MI)
    • 내용: 변수는 #{변수명} 형식 (예: #{고객명}님 주문이 완료되었습니다.)
    • 버튼: 웹링크(WL), 앱링크(AL), 배송조회(DS), 채널추가(AC) 등
    • 글자수: 기본형 최대 1,000자 (변수 포함)
  3. 카카오 검수 신청 → 심사 기간: 약 1~2 영업일
    • 반려 사유: 광고성 문구, 변수 위치 부적절, 맞춤법 오류 등
    • 반려 시 수정 후 재신청 가능
  4. 승인 완료 → templateCode 확정
주의: 발송 시 메시지 내용이 승인된 템플릿과 정확히 일치해야 합니다. 변수 부분만 다를 수 있으며, 그 외 텍스트가 다르면 3016 에러 발생.

  1. 서버 환경변수 설정: 
    SEJONG_SENDER_KEY=발급받은senderKey(40자)
SEJONG_TEMPLATE_CODE=승인된templateCode
  2. TalkPC 클라이언트에서 발송 방법 → "알림톡 (세종)" 선택
  3. 메시지 내용을 승인된 템플릿과 동일하게 작성 (변수만 실제 값으로 대체)
  4. 발송 테스트 → 결과 확인
알림톡 실패 시 대체발송: SMS 또는 LMS로 자동 재발송 가능 (resendType 설정). 대체발송 시 별도 과금됩니다.
RCS 사용 가이드
RCS는 RCS 비즈센터 가입 → 브랜드 등록 → 챗봇(발신번호) 등록 → 세종텔레콤 연동 절차가 필요합니다. 전체 소요 기간: 약 5~10 영업일

  1. RCS 비즈센터 접속 → 회원가입 (사업자 정보 필요)
  2. 브랜드 등록 신청
    • 사업자등록증, 브랜드명, 로고 이미지 등 제출
    • 심사 기간: 약 2~5 영업일
  3. 브랜드 승인 완료 → brand_key 발급

  1. RCS 비즈센터 → 챗봇 관리 → 새 챗봇 등록
  2. 발신번호 입력 (사전 등록된 번호만 가능)
  3. 챗봇 승인 완료 → chatbotId = 등록된 발신번호
RCS의 chatbotId는 등록된 발신번호입니다. 예: 01097804669

  1. 세종네트웍스 담당자에게 RCS 사용 신청
  2. RCS 비즈센터에서 세종텔레콤을 대행사로 등록
  3. 세종 측에서 RCS 발송 기능 활성화
  4. 서버 환경변수 설정: 
    WIDESHOT_CHATBOT_ID=등록된발신번호
